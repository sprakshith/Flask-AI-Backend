1. Todo List Management

    1.1. As a user, I want to create, edit, and delete todo lists so I can organize my tasks.

    Acceptance criteria:
        - User can create a new todo list by entering a title.
        - User can edit the title of an existing todo list.
        - User can delete a todo list and all associated tasks.

    1.2. As a user, I want to add, edit, and delete tasks within a todo list so I can manage my tasks efficiently.

    Acceptance criteria:
        - User can add a new task to a todo list by entering a task description.
        - User can edit the description of an existing task.
        - User can delete a task from a todo list.
